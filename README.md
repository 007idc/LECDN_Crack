## 功能

本仓库可帮助你开心的试用Lecdn

## 使用方法

安装完Lecdn后从仓库中下载修改过的lecdn-master.patch放到Lecdn目录下

进入Lecdn目录执行指令

```bash
chmod +x lecdn-master.patch
```

完成后执行下面命令修改Hosts

